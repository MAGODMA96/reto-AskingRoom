## Instalacion

Para visualizar el reto ejecuta los siguientes comandos:

`*Primer comando`
```
cp .env.example .env
```

`*segundo comando`
```
Composer install
```

`*tercer comando`
```
npm install
```

`*cuarto comando`
```
Php artisan migrate --seed
```
